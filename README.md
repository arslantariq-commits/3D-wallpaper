# NexBot Interactive Live 3D AI Wallpaper

A futuristic interactive 3D robot character built with **Spline** and **Face-api.js** that tracks your face using your webcam.

## Features
- Real-time AI face tracking directly inside the browser.
- Interactive NexBot 3D model that looks at you wherever you move.
- Fully compatible as a Desktop Live Wallpaper via **Lively Wallpaper**.

## How to use on Desktop (Lively Wallpaper)
1. Turn on GitHub Pages for this repository to get a live URL.
2. Download **Lively Wallpaper** from Microsoft Store.
3. Click **Add Wallpaper** and paste your GitHub Live URL.
4. Enjoy your futuristic interactive desktop background!
